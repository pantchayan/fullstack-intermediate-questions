Q3 What are the advantages of using this code in a html page ?

<meta name="viewport" content="width=device-width, initial-scale=1">

a) It helps in better seo.
b) It tells webpage to use whole viewport of the screen
c) width=device-width is used to override defaults in mobile screens.
d) b & c
Ans : 
d) b & c
